# Secure Multi-Cluster Kubernetes Platform on GCP

[![Terraform Validate](https://github.com/RahmaHaouas/gcp-secure-kubernetes-platform/actions/workflows/terraform-validate.yml/badge.svg)](https://github.com/RahmaHaouas/gcp-secure-kubernetes-platform/actions/workflows/terraform-validate.yml)

A hardened multi-cluster Kubernetes platform on Google Cloud, built from an explicit threat model.

## Overview

This project designs, deploys, and secures a production-grade Kubernetes platform on Google Cloud. Every security control maps to an identified threat and is validated by a reproducible attack.

## Planned Stack

`GCP` · `GKE` · `Terraform` · `Istio` · `ArgoCD` · `Kyverno` · `Trivy` · `GitHub Actions` · `Prometheus` · `Grafana`
